## 역할별 푸시 메세지

>front-end : 임준섭 back-end : 이승범 

`선택한 역할 보기` : 현재 선택되어있는 역할을 보여줍니다.

`역할 선택하기` : 역할을 선택합니다.

`새로운 역할 추가하기` : 역할 선택 옵션에 새로운 역할을 추가합니다.

`나의 역할 초기화하기` : 현재 선택되어있는 역할을 초기화합니다.

`메세지 보내기` : 역할별 메세지를 보낼 수 있습니다.
